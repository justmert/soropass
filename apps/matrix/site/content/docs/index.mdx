  <Card title="Proven on-chain">

  </Card>
  <Card title="Headless-first UI">

    token-driven styled layer. Re-theme by overriding CSS tokens only.
  </Card>
  <Card title="Living compatibility matrix">
    A dated, diffable WebAuthn/passkey matrix merged from MDN BCD + live feature-detection +
    virtual-authenticator CI — sourced, not asserted from memory.
  </Card>
  <Card title="Security, documented">
    Low-S malleability, challenge/replay, RP-ID binding and recovery are written up in the threat
    model, each tied to a battle-tested anchor.
