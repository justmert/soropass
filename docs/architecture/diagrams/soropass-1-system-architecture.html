🦊Soropass — System Architecture
A device passkey is the signer of a Soroban smart account. The whole pipeline runs client-side against public soroban-rpc — no Soropass server, no API key, no database in the user's auth path. Managed adapters are optional drop-ins.
Client · browser / wallet no backend required
